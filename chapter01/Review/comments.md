프로그램의 시작은 main()
#include <iostream>은 입출력 도구 사용
cout은 화면 출력
문장 끝은 ;

변수는 데이터를 저장하는 이름 있는 공간
자료형은 저장 방식의 약속
string, int의 역할 구분

조건문은 프로그램의 흐름을 나누는 구조
비교 연산자 <=
참/거짓 판단

반복문은 같은 작업을 여러 번 수행
게임의 턴 개념과 연결

함수는 작업을 이름 붙여 묶는 것
재사용 가능
코드가 짧아지고 의미가 분명해짐

관련 데이터와 기능을 하나로 묶음
자료구조에서 노드(Node), 리스트(List), 트리(Tree)도 결국 이런 식으로 구현
객체지향을 자료구조와 연결
